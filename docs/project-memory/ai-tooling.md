# AI Tooling

## Usage Guidelines

AI tools, such as Jules, can be used to aid in the development process for writing and reviewing code.

## Restrictions

1. **No Auto-Merges**: Jules should create branches and Pull Requests (PRs) but is strictly forbidden from merging automatically.
2. **Mandatory Reviews**: All changes suggested or made by AI tools must be thoroughly reviewed by a human engineer before they can be accepted.
3. **Restricted Access**: The repository should remain securely accessible only to the AI Companion team.

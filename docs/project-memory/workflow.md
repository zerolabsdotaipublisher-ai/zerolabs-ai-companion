# Workflow

## Project Context

This is the **AI Companion** project.

## Development Process

- **Jira-Driven**: All development work is driven by Jira tasks.
- **Repository Access**: Repository access should remain limited to the AI Companion team only.

## AI Collaboration Rules

- AI tools (such as Jules) may assist in writing code, but **all changes must be reviewed** by a human engineer.
- **Jules constraints**: Jules should create branches and Pull Requests (PRs) only. Jules must **not** merge automatically.
- **QA Requirements**: Manual QA is required before any merge.

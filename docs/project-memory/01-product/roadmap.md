# Roadmap

## Current phase

AI Companion has completed platform and authentication/identity foundation through AIC-206.

## Next likely product phases

The next phase should begin moving from foundation into the actual daily companion MVP.

Potential future stories:

1. Dashboard refinement for companion MVP entry point
2. Daily suggestion screen
3. Suggestion data model
4. Suggestion do/skip/ignore behavior
5. Lightweight capture flow
6. Memory summary generation
7. Timeline view
8. Profile-driven personalization for suggestions

## Future scale phase

Later, the system may evolve into:

- Pattern recognition
- Semantic memory search
- Life insights
- Weekly/monthly summaries
- Travel companion mode
- Story generation
- AI Autobiography integration
- ZeroFlow orchestration and usage tracking

## Deferred technical capabilities

- OpenAI integration is planned but not yet active in product flow.
- Qdrant is optional and deferred until retrieval has real value.
- Wasabi/shared storage is a future scale option.
- ZeroFlow should remain outside the product until orchestration/cost tracking is needed.

## AI Context

Do not treat roadmap items as implemented. Roadmap items are future direction only.
